## 결정 내용

MVP는 **뮤지션 포트폴리오 + 셀프온보딩** 중심으로 구축한다.

---

## 결정 배경

### 탈락한 옵션들

**Option A (공연 캘린더 퍼스트)**
- 재즈클럽 스크래핑으로 공연 정보 채움
- 탈락: 재즈피플과 차별점 없음. "Insight Navigator" 컨셉 파괴

**Option C (도슨트 노트 퍼스트)**
- 창업자가 직접 15개 깊은 노트 작성
- 탈락: 창업자가 지속적으로 글을 쓸 자신이 없음

### Option B 채택 이유

1. **뮤지션에게 실질적 가치**: "디지털 이력서" 페이지는 뮤지션 본인에게도 필요
2. **자발적 참여 가능**: 재즈 뮤지션들은 인스타그램에 공연 홍보글을 자체 작성하는 습관이 있어 데이터 입력 부담이 낮음
3. **컨셉 유지**: 관계도와 포트폴리오는 DAZZ 핵심 기능 그대로
4. **현실적 인맥 활용**: 창업자(서울예대 재즈피아노 전공)의 현업 뮤지션 지인 네트워크

---

## 실행 방법

1. 프론트엔드 완성 후 링크를 지인 뮤지션들에게 공유
2. "여기서 너를 홍보할 수 있어" 방식으로 온보딩
3. 공연이력이 많으면 최근 N개월치만 요청

### 인맥풀 활용 현실적 범위

| 요청 | 가능 여부 |
| --- | --- |
| 프로필 사진 + 이름 + 악기 | ✅ |
| 최근 공연이력 (인스타 홍보글 기반) | ✅ |
| "이 분이랑 같이 연주했지?" 확인 | ✅ |
| 2019년부터 전체 공연이력 정리 | ❌ |
